# Spectre Documentation Website

This repository contains the documentation website for [Spectre](https://github.com/moolen/spectre) - a Kubernetes observability platform that monitors resource changes and stores them in a graph database for analysis and visualization.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

The site is automatically deployed to GitHub Pages when changes are merged to `main`.
